# ReactApp3 Task Manager

A simple full-stack task manager built with React (Vite) and Express.

## Features

- View tasks from a mock API (in-memory data)
- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start backend API:

```bash
npm run server
```

3. In another terminal, start frontend:

```bash
npm run dev
```

- Frontend: http://127.0.0.1:5173
- API: http://127.0.0.1:4000/api/tasks
